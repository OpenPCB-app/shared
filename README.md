# @openpcb/rendering-core

Pure render-model builders and geometry primitives for the OpenPCB ecosystem.
Zero React/Three.js deps — runs in browsers, Node, and Bun.

## Exports

- `buildSymbolRenderModel`, `buildFootprintRenderModel`
- IPC-7351B footprint generator (`ipc7351b/*`)
- Parametric generators: pin header, mounting array, screw terminal
- Geometry/bounds helpers
- KLC constants and snap-grid defaults
